#### 清除

```
hexo clean
```

#### 生成静态文件

```
hexo g 或者 hexo generate
```

#### 启动本地服务

```
hexo s 或者 hexo server
```

#### 部署到远端服务

```
hexo d 或者 hexo deploy
```

#### 可以组合使用

```
hexo clean & hexo g & hexo s
```

```
hexo clean & hexo g & hexo d
```

#### 创建本地md文件（位置为：../source/_posts/xxx.md）

```
hexo new xxx 或者 hexo n xxx
```
